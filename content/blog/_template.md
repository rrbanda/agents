# Blog Post Template

Use this template when creating a new blog post.

## File Naming Convention

`YYYY-MM-DD-kebab-case-title.md`

Example: `2024-12-14-introduction-to-agentic-ai.md`

## Front Matter

```yaml
---
title: "Your Blog Post Title"
subtitle: "Optional subtitle or hook"
date: "YYYY-MM-DD"
author: "Author Name"
status: "draft|published"
tags: ["tag1", "tag2", "tag3"]
category: "fundamentals|intermediate|advanced|news"
featured: false
estimated_reading_time: "X min"
cover_image: "./images/cover.jpg"
---
```

## Content Structure

```markdown
# Title

## Introduction
Hook the reader. What problem are you solving? Why should they care?

## Section 1: Background
Provide context and foundational information.

## Section 2: Main Content
Deep dive into the topic. Use:
- Code examples
- Diagrams
- Real-world scenarios

## Section 3: Practical Application
How can readers apply this knowledge?

## Conclusion
Summarize key points and provide next steps.

## References
- [Reference 1](https://...)
- [Reference 2](https://...)
```

## Best Practices

1. **Length**: 1,500-3,000 words for in-depth articles
2. **Images**: Store in `content/resources/assets/images/blog/`
3. **Code**: Use fenced code blocks with language hints
4. **Links**: Prefer relative links for internal content
5. **SEO**: Include relevant keywords naturally
